# POPcorn-Clicker
Click on the flying popcorn kernels to earn points!

HOW TO PLAY
WELCOME TO POPCORN CLICKER!

Click the popcorn kernels flying out of the machine to score points.

Each kernel is worth 5 points.

DON'T CLICK the Burnt kernels, or else you'll instantly lose!

Combo: 2 correct clicks in a row -> 2x multiplier
Missing following clicks resets combo

The rare Gold kernels are worth 15 points.

Have fun!
